
# Refactoring Prop Drilling to Context

## Description

I created an app using Vite and React that refactors prop drilling using the React Context API.

The app has 4 connected components:

- App.jsx

- Dashboard.jsx

- Sidebar.jsx

- UserProfile.jsx

The user data is shared globally using Context instead of passing props through multiple components.

## Technologies Used

- React

- Vite

- JavaScript

- Context API

## How to Run

```bash

npm install

npm run dev
